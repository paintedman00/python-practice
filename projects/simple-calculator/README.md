# Simple Calculator

A basic command-line calculator written in Python.

## Features

*   Addition
*   Subtraction
*   Multiplication
*   Division

## Usage

Run the `main.py` script from your terminal.
The calculator will prompt you to enter two numbers and an operation.

```bash
python main.py
```

## Examples

```
Enter first number: 10
Enter second number: 5
Enter operation (+, -, *, /): +
Result: 15.0
```
