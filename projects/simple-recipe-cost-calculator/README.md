# Simple Recipe Cost Calculator

This is a simple command-line application that helps you calculate the cost of a recipe based on the ingredients and their quantities.

## Usage

1.  Run the script: `python main.py`
2.  Follow the prompts to enter the recipe name, ingredients, their quantities, and costs.
3.  The program will calculate the total cost and cost per serving.

## Example

```
python main.py
```
