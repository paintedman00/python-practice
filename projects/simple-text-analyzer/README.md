# Simple Text Analyzer

A simple command-line tool to analyze text and provide basic statistics like word count, character count, and average word length.

## Usage

Run the script using `python main.py` and provide the text you want to analyze as input.

## Example

```
python main.py
Enter text: This is a simple example.

Word Count: 5
Character Count: 24
Average Word Length: 4.0
```
