# Simple Grade Averager

A simple command-line tool to calculate the average grade from a list of scores.

## Usage

Run the script and enter the grades one by one, separated by spaces.  When finished, enter 'done'. The script will then calculate and display the average grade.

## Example

```
python main.py
Enter grades (separated by spaces, type 'done' when finished): 85 90 78 92 done
Average grade: 86.25
```
