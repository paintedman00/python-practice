# Simple Shopping List

A simple command-line application to manage a shopping list.

## Usage

Run `python main.py` and follow the prompts.

*   `add <item>`: Adds an item to the shopping list.
*   `remove <item>`: Removes an item from the shopping list.
*   `list`: Displays the current shopping list.
*   `clear`: Clears the entire shopping list.
*   `exit`: Exits the application.

## Example

```
> python main.py
Welcome to the Simple Shopping List!
> add milk
Added milk to the shopping list.
> add eggs
Added eggs to the shopping list.
> list
Shopping List:
1. milk
2. eggs
> remove milk
Removed milk from the shopping list.
> list
Shopping List:
1. eggs
> exit
Goodbye!
```
