# Simple Palindrome Checker

A simple command-line tool to check if a given string is a palindrome.

## Usage

Run `python main.py` and enter a string when prompted. The program will output whether the string is a palindrome or not.

## Example

```
python main.py
Enter a string: racecar
'racecar' is a palindrome.
```

```
python main.py
Enter a string: hello
'hello' is not a palindrome.
```
