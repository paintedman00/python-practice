# Simple Vowel Counter

A simple command-line tool to count the number of vowels in a given string.

## Usage

Run the script from the command line:

```bash
python main.py
```

The script will prompt you to enter a string. After you enter the string, it will print the number of vowels (a, e, i, o, u) found in the string.

## Example

```
Enter a string: Hello World
Number of vowels: 3
```
