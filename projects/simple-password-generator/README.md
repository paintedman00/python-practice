# Simple Password Generator

This is a simple command-line password generator written in Python. It allows the user to specify the desired length of the password and generates a random password containing letters, numbers, and symbols.

## Usage

Run the `main.py` script from the command line. The script will prompt you to enter the desired password length.  Enter a valid integer length to generate your password. An invalid input will produce an error message.

Example:
```bash
python main.py
```
