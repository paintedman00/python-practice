# Simple Color Mixer

A simple command-line tool to mix two colors and display the resulting color.

## Usage

Run `python main.py` and follow the prompts to enter two colors. The program will output the mixed color.

## Supported Colors

The program supports the following colors: red, green, blue, yellow, cyan, magenta, white, black.

## Example

```
python main.py

Enter the first color: red
Enter the second color: blue

Mixed color: magenta
```
