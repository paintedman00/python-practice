# Simple Alarm Clock

A simple command-line alarm clock written in Python.

## Usage

Run `python main.py` and follow the prompts to set the alarm time.

The alarm will print a message when the specified time is reached.

## Example

```
$ python main.py
Enter the alarm time (HH:MM): 07:30
Alarm set for 07:30
...
Alarm ringing! It's 07:30
```
