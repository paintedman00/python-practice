# Simple Mad Libs Generator

A simple command-line Mad Libs generator written in Python.

## Usage

Run the script `main.py` and follow the prompts to fill in the blanks.

```bash
python main.py
```

## Example

```
Please enter an adjective: happy
Please enter a noun: dog
Please enter a verb: run
Please enter an adverb: quickly

Here's your Mad Lib:

The happy dog likes to run quickly.
```
