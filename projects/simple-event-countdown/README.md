# Simple Event Countdown

A simple command-line tool to calculate the days remaining until a specified event.

## Usage

Run `python main.py` and follow the prompts to enter the event name and date (YYYY-MM-DD).

The program will then output the number of days remaining until the event.

## Example

```
python main.py
```

```
Enter the event name: Birthday
Enter the event date (YYYY-MM-DD): 2024-12-25

Days until Birthday: 205
```
