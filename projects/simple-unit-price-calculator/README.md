# Simple Unit Price Calculator

This is a simple command-line tool to calculate the unit price of a product, given its total price and quantity.

## Usage

Run the script `main.py` and follow the prompts to enter the product's name, price, and quantity. The script will then calculate and display the unit price.

## Example

```
$ python main.py
Product Name: Apples
Price: 2.50
Quantity: 5

The unit price for Apples is: $0.50 per unit
```
