# Simple Random Number Guessing Game

A simple command-line number guessing game written in Python.

## How to Play

1.  Run the `main.py` script.
2.  The game will generate a random number between 1 and 100 (inclusive).
3.  You will be prompted to guess the number.
4.  The game will tell you if your guess is too high or too low.
5.  Keep guessing until you guess the correct number.
6.  The game will tell you how many guesses it took.

## Usage

```bash
python main.py
```
