# Simple Book Catalog

A simple command-line tool to manage a small book catalog.  Allows you to add books, list books, and search for books by title.

## Usage

Run `python main.py` to start the program. Follow the on-screen prompts.

## Features

*   Add a new book to the catalog (title, author).
*   List all books in the catalog.
*   Search for a book by title (case-insensitive).
